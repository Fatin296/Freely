## üì• Installation Guide (Manual Install)

Since this extension is currently in development (or strictly for private use), you must install it using Chrome's Developer Mode.

### ‚ö†Ô∏è Important Pre-Install Step
Before installing, ensure you have a clean folder structure:
1.  Download the ZIP file.
2.  Unzip it to a folder on your computer.
3.  **Crucial:** Open the unzipped folder and **delete the `.gitattributes` file** if it exists.

### How to Install in Chrome
1.  Open Google Chrome.
2.  Type `chrome://extensions/` in the address bar and press Enter.
3.  Toggle **Developer mode** on in the top-right corner.
4.  Click the **Load unpacked** button (top-left).
5.  Select the folder you unzipped (make sure you select the folder containing `manifest.json`).
6.  The extension is now installed! You should see the **Freely** icon in your toolbar.
